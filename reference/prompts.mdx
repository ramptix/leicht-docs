## `def` get_prompt

```python
def get_prompt(
  name: str,
  *, 
  no_cache: bool = False, 
  **kwargs: str
) -> str
```

Fetch or read a cached prompt from [ramptix/preprompted-data](https://github.com/ramptix/preprompted-data). Prompts will be saved to the `.preprompt/` directory.

<ParamField path="name" type="str">
  Name of the prompt. See the Preprompt (preprompted-data) [Wiki](https://github.com/ramptix/preprompted-data/wiki) for verified prompts, or refer to the [Discussions page](https://github.com/ramptix/preprompted-data/discussions/1) for prompts by the community.
</ParamField>

<ParamField path="no_cache" type="bool" default="False" required={false}>
  Don't cache the prompt?
</ParamField>

<ParamField path="**kwargs" type="str" required={false}>
  Additional keyword-only arguments. These will be parsed as "{key}={value}" pairs.
  For instance, a prompt like:

```
Pizza tates {adjective}
```

Given keyword arguments (`**kwargs`): `adjective="delicious"`, returns a modified prompt:

```
Pizza tastes delicious
```

</ParamField>
